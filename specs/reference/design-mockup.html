sterm 扩展设计 — 技术概要

图层终端（主） + 侧边面板（可折叠） + 命令面板（CmdK 覆盖层）
设计体系Linear Design Tokens（暗色画布 #08090a、510 字重、8px Grid）
数据存储全部本地：命令 JSON → ~/.sterm/snippets.json · 速查 JSON → ~/.sterm/cheats/*.json · 设置 → localStorage
不需要后端改的是 app.js + index.html，不动 server.js / lib/
快捷键CmdK 命令面板 · Cmd⇧S 收藏当前命令 · Cmd⇧L/P 快速插入
核心交互命令按钮 → ws.send() 模拟终端输入 · 搜索 → 前端模糊过滤 · 速查 → 卡片内嵌 iframe/预览片段
依赖零新增 npm 包（fuse.js 可选项，用于模糊搜索命令）
